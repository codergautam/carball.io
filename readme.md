# carball.io

## how to run on a server
1. pnpm i
2. npm run build
3. node .

Running on port 3000

## development

running client in dev mode:
1. pnpm i
2. cd client
3. npm run watch

Running server in dev mode:

Now open a new terminal while that's running
1. node .

Running on port 3000

Make sure to restart `node .` whenever you make a server change

Enjoy!
